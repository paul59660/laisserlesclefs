Un paquet d'informations doit être transmis, le moment venu, aux intéressé(e)s. Lesquelles ? 

Elles sont de différentes sortes: 

1. personnelles: 

- les femmes, les hommes  
- leur identité
- leurs caractéristiques    

2. institutionnelles:

- les structures dont ils font parties
- les contrats   

3. matérielles:

- les biens ( matériel et financier )
- les engagements ( abonnements )    

4. des remarques utiles:

- ce qui peut servir
